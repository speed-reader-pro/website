Website Analytics

Our website uses Umami, a privacy-focused analytics tool, to collect anonymous usage statistics (page views, referral sources, browser type, OS). Umami does not use cookies, does not collect personal data, and does not track users across websites.

macOS Application

All your reading content (PDF, TXT, Markdown, EPUB, FB2, DOCX, images, URLs, clipboard text, and screen captures) is processed locally on your device using native macOS frameworks. We do not collect, transmit, or store any of your reading content.

The app uses Sentry for anonymous crash reporting and TelemetryDeck for anonymous usage analytics. These services collect only non-personal technical data: app events (e.g., reading started, settings changed), file format types (not filenames or content), OS version, and error logs. IP addresses are not collected. No reading content, URLs, filenames, or personal data is sent.

The app uses Sparkle to check for updates via a periodic request to our update server. No personal data is transmitted.

Screen capture, clipboard, and file access are used only when you explicitly initiate these actions.
